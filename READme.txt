УСТАНОВКА:

1) Запустить Settup.exe;
2) Установить программу;
3) Установить python2 в зависимости от разрядности;
4) Создать задание в планировщике заданий Windows в ручном режиме или с помощью bat файла:
	4.1) В ручном режиме выбрав:
		4.1.1) Действие - запуск программы dbChecker.py, рабочая директория - полный путь к директории mailChecker;
		4.1.2) Тригеры - каждый день, каждые 5 минут;
	4.2) С помощью bat файла:
		4.2.1) Открыть taskCreator.bat в текстовом редакторе;
		4.2.2) После флага /xml указать полный путь к файлу mailCheckerTask.xml находящемся в папке mailChecker;
		4.2.3) Открыть файл mailCheckerTask.xml;
		4.2.4) В поле  <Command> установить полный путь к файлу dbChecker.py в дирректории mailChecker;
		4.2.5) В поле  <WorkingDirectory> установить полный путь к директории mailChecker;
		4.2.6) Запустить файл taskCreator.bat от имени Администратора (если выполнено успешно taskCreator.bat можно удалить)
		4.2.7) Проверить в планировщике заданий Windows, что задча mailChecker имеет состояние Готово;
5) Открыть файл Settings.txt:
	5.1) Указать полный путь к файлу mailDeliverDatabase.db в поле DATABASE_LOCATION;
	5.2) Указать в полях EMAIL и PASSWORD Settings.txt данные для входа в почту;
	5.3) Указать в поле SMTP_SERVER название smtp сервера(самые частые примеры приведены ниже в разделе настройки);
	5.4) Для более детальной настройки смотри раздел Настройки
=======================================================================================================
РАБОТА С ПРОГРАММОЙ:

	ОТПРАВИТЬ ПИСЬМО:
	1) Открыть mailDeliver.exe
	2) Нажать добавить
	3) Создать и сохранить письмо
	4) Письмо будет отправлено в заданное время с погрешностью в 5 минут
	5) Логи сохраняются в файле logs.log в директории mailChecker 
		(при установке на рабочем столе создаются ярлык для открытия данного файла)
	ДОБАВИТЬ ГРУППУ:
	1) Добавить -> группы адресов
	2) Заполнить поля названия группы и email адресов, входящих в данную группу (разделитель - ", " - запятая пробел)
	СОЗДАНИЕ ПИСЬМА:
	1)Добавить
	2)Заполнить поля:
		2.1) Кому - ввести email адрес или выбрать группу справа от поля и нажать кнопку добавить группу
		2.2) Тема - заголовок сообщения
		2.3) Сообщение - тело сообщения
		2.4) День отправления:
			2.4.1) Дата - выбор конкретной даты отправки сообщения
			2.4.2) День недели - выбор дня недели в который будет происходит отправка
		2.5) Время отправления - время, в которое сообщение будет отправлено
		2.6) Прикрепить файл - файлы, прикрепленные к данному сообщению:
			2.6.1) При BROWSE_TYPE равном FILE выбирается конкретный файл для отправки
			2.6.2) При BROWSE_TYPE равном DIRECTORY выбирается директория из которой будут отправлены все файлы
	3) Кнопка добавить для добавления сообщения в базу данных
	4) Кнопка отменить для закрытия страницы создания сообщения
=======================================================================================================
НАСТРОЙКИ:

EMAIL - емайл с которого будет отправлено письмо
PASSWORD - пароль от почты с которой будет отправлено письмо
SMTP_SERVER - имя smtp сервера ниже приведены примеры наиболее частых smtp серверов
				smtp-mail.outlook.com 	for 	outlook
				smtp.gmail.com        	for 	gmail
				smtp.yandex.ru        	for 	yandex
DATABASE_LOCATION - полный путь к базе данных
DEFAULT_FILE_PATH - папка, которая будет открываться при нажатии на кнопку обзор
DELETE_AFTER_SENDING - удалять или нет запись их базы данных после отправки, 
						принимает значения (YES и NO)
BROWSE_TYPE - 	режим выбора файлов для отправки.
				Варианты: 	FILE - выбор файла для отправки
							DIRECTORY - выбор директории с файлами
TIME_DELTA - время разницы между временем отправки файла и нижней и верхней границами проверки времени
LOG_FILE_ATTACHED_FILES_SEPARATOR - символ-разделитель между прикрепленными файлами в логах 
									(для отображения файлов в столбик используйте \n).
